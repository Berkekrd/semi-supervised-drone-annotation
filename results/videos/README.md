# Tracking-Videos

Dieser Ordner enthält die qualitativen Tracking-Ergebnisse der Pipeline: Beispielvideos, in denen die mit dem YOLO-Modell erkannten Objekte sowie die per ByteTrack vergebenen Track-IDs als Overlay über das Originalvideo gezeichnet sind. Zusätzlich werden die GPS-Daten des jeweiligen Drohnenflugs einbezogen.

> **Hinweis:** Die Videodateien (`*.mp4`) sind aufgrund ihrer Größe (mehrere GB, 4K-Auflösung) **nicht** im Git-Repository enthalten (siehe [`.gitignore`](../../.gitignore)). Sie werden extern gehostet und sind über die untenstehenden Links erreichbar. Im Repository verbleiben pro Video nur der GPS-Tracking-Report (`*.csv`), die Zusammenfassung (`*.txt`) und die Beschreibung (`Readme.md`).
